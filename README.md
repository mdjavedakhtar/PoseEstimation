# PoseEstimation
WinUI3 C# Human body pose keypoint find from webcam feed using TensorFlow model and OpenCVSharp4
Thanks to https://github.com/shimat/opencvsharp for writing the OpenCVSharp4 library it was really helpful

##Technology Used
.Net WinUI3
C#
OpenCV
OpenCVSharp4
